This page lets you download Quelea for your preferred platform directly. All releases require a working Java 7 installation which you can download here (select JRE 7, you don't need the JDK.) Quelea is licensed under the GPLv3 and the download includes the full Java source code.

Older releases, any beta / release candidates and technical material can be found on the Google code page.

Current version: 0.5

Windows (XP and above) Ubuntu (11.10 and above) Cross platform
Where's Quelea for Mac?

We're working on an official Mac release of Quelea, but right now Java 7 isn't officially supported which makes it difficult to provide a Mac distribution. If you wish you can download the cross platform install and try that, but you must have Java 7+ (not 6) installed, otherwise it won't work.
