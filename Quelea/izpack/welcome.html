Quelea installation

This will install Quelea on your system. IMPORTANT NOTE: From 2022.0 onwards, you MUST have Java 11 or newer installed on your system, or Quelea will not run. Java 8 is no longer supported.

If you have any issues with the installation or use of Quelea, please see the website or feel free to start a discussion topic asking for help.

We hope you enjoy using the software!
